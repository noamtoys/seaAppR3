# COBMINDEX App Screenshots

## How to Add Your Images

Please save your app screenshot images in this folder with the following names:

### Main Images:
1. **hero-mockup.png** - The main hero/overview image showing multiple screens or app showcase
2. **mood-tracking.png** - Screenshot of the mood and symptom tracking interface
3. **progress-charts.png** - Screenshot showing progress charts and analytics
4. **exercises.png** - Screenshot of the exercise/practice library

### Additional Screens:
5. **dashboard.png** - Main dashboard view
6. **calendar.png** - Practice calendar/schedule view
7. **guidance.png** - Guidance or instructions module
8. **multiple-screens.png** - Overview showing multiple app screens together

## Image Requirements:
- Format: PNG (recommended) or JPG
- Recommended width: 1080px - 1500px
- Keep aspect ratio for mobile screenshots
- Use clear, high-quality images
